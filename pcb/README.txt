File extensions:
Eagle schematic: .sch
Eagle board: .brd
Eagle library: .lbr

Gerber layers:
.crc: component side solder cream frame
.plc: component side silkscreen
.stc: component side solder mask
.cmp: component side copper
.ly2: inner layer copper (layer 2)
.l15: inner layer copper (layer 15)
.sol: solder side copper
.sts: solder side solder mask
.pls: solder side silkscreen
.crs: solder side solder cream frame

Drill files:
.drd: Excellon drill file
.dri: Drill station info file
